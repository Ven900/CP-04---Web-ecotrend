##  Descrição

O **EcoTrend** é um e-commerce de produtos sustentáveis, desenvolvido com foco em práticas ecológicas e na aplicação de conceitos modernos de desenvolvimento web com JavaScript.

A aplicação permite a visualização dinâmica de produtos, filtragem por categoria e simulação de compras através de um carrinho interativo com persistência de dados.

---

##  Funcionalidades

* Carregamento de produtos via arquivo JSON
* Carrinho de compras com controle de quantidade
* Persistência de dados utilizando localStorage
* Filtro dinâmico de produtos por categoria
* Sistema de avaliações visuais
* Exibição de produtos com desconto
* Interface moderna, organizada e responsiva

---

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)

---

## Como Executar

1. Baixar ou clonar o repositório
2. Abrir o arquivo **index.html** em um navegador
3. Navegar pelos produtos e utilizar o carrinho de compras
